# Chat 58

这是一个面向 `58 商家中心 -> 人才管理` 页面的自动化工具。

程序作用：

1. 查找当前 Windows 上安装的 `msedge.exe`
2. 使用程序同目录下的独立 `edge_profile` 资料目录启动或接管 Edge
3. 打开 `https://employer.58.com/main/jobmanage`
4. 如果进入登录页，则等待用户手动登录
5. 进入列表页后，每 10 分钟刷新一次并重复执行筛选流程
6. 从当前页筛选 `18~55 岁男性`
7. 当操作列是 `在线沟通` 时自动点击
8. 到达设定运行时长后自动退出；也支持永久执行
9. 可记忆“到时自动关闭浏览器”开关，按当前设置决定是否关闭项目专用 Edge

## 运行

```powershell
uv run main.py
```

启动后会先显示运行时长菜单：

- `1~8 小时`
- `0` 为 `永久执行`
- `9` 为切换“到时自动关闭浏览器”，并记住上次选择

如果跳到登录页，直接在弹出的 Edge 窗口里登录即可，程序会自动继续。

如果运行中 Edge 被关闭、崩溃或调试连接断开，程序会自动重连，并优先从本轮上次中断的候选人继续。

## 点击规则

程序点击 `在线沟通` 时按以下规则处理：

- 点击后按钮变成 `继续沟通`，视为成功
- 点击后出现右下角弹框，自动关闭，并视为成功
- 点击后无变化，最多重试 3 次，再跳过

## 年龄与性别判断

- 性别使用页面接口中的 `sex` 字段判断
- 年龄使用页面的自定义字体编码动态解码后判断
- 如果个别记录年龄无法解码，会跳过并打印到日志中

## 打包 exe

```powershell
.\build.ps1
```

打包结果默认在 `dist\edge_58_launcher.exe`。

## 使用说明

- 目标机器需要已经安装 Microsoft Edge
- 程序会在脚本或 `exe` 同目录自动创建 `edge_profile`
- `edge_profile` 会保留登录态、Cookie 等独立浏览器数据
- 程序会自动选择可用的 CDP 端口，避免与本机已有 Edge 调试实例冲突
- 如果运行中浏览器连接断开，程序会自动重新接管 Edge，并按当前轮次的断点继续
- 到达设定运行时间后，是否自动关闭当前 `edge_profile` 对应的 Edge，取决于菜单中的开关设置
